Determine Website's Server Spece

Written by Chiayo Lin | AUG 04, 2014 | Taipei, Taiwan


The following commands will print what web server the website is running on.

curl -I# www.foobar.com | grep 'Server:' | awk '{ print $2 }'

There are three commands above, each command is separated by a “|” (vertical slash). Among of these three commands include, curl, grep, and awk. We get information from a server by using curl with two command line option, -I and -#. Use man curl to see what these two options do if you want to. However, this command will prints a bunch of information that we don’t need for now. Therefore, we use classic UNIX command, grep (man grep for more information) to search for pattern “Server:” then prints it. It makes terminal output more readable. Finally, we use awk to remove the string, "Server:", so the output will be only the name of web server that the website is running on.


Example


Input:

curl -I# www.linux.org | grep 'Server:' | awk '{ print $2 }'

Output:

Apache

Done!
