Install Package from AUR

Written by Chiayo Lin | JUN 13, 2014 | Saigon, Vietnam


Arch User Repository (AUR) is a community-driven repository for Arch Linux users. You can find packages that haven't been officially supported by Arch Linux on AUR, such as Google Chrome, Dropbox, and Libre Office. AUR contains package descriptions (PKGBUILDs) that allow you to compile a package from source with makepkg and then install it via pacman.


Getting Started


  1. Install build essentials:
    2. # pacman -S --needed base-devel
  2. Download the tarball of the package you want from AUR. Then switch your working directory to where your tarball is:
    3. $ cd /path/to/directory/
  3. Extract the tarball that you downloaded by uing tar:
    4. $ tar -zxvf FOOBAR.tar.gz
  4. cd into extracted folder:
    5. $ cd /path/to/FOOBAR/
  5. Make the package:
    6. $ makepkg -Acs
    Do not run makepkg as root!
  6. Finally, install the package with filetype pkg.tar.xz
    # pacman -U FOOBAR.pkg.tar.xz

Done!
